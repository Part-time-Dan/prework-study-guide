Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • Elements nested inside other elements are called child elements. The element that holds the children is called the parent element.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • An ELEMENT refers to the entire element from opening tag to closing tag, whereas a TAG refers only to what appears inside the angle brackets.

CSS

  • A margin indicates how much space we want around the outside of the element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • For passphrase, open GitBash and in CL enter $ eval "$(ssh-agent -s)" to start ssh-agent. Then use command $ ssh-add to safe passphrase for session.
  • git status: checks what branch we are currently on.
  • git checkout -b (branch-name): creates a new branch and switches to it.
  • git checkout (branch-name): navigates to that branch.
  • git add -A: adds any changed files in the working directory to the staging area of Git.
  • git commit -m "(your comments here)": commits staged changes to the repo and includes a message.
  • git pull origin main: Makes sure the current branch is up to date before opening PR (pull request)
  • git push origin (feature branch-name): Pushes all our changes to Git to be reviewed and merged.
